## Usage

<CopyCode>
  {`import { ImageText } from "@/components/ui/image-text";

export default function Example() {
  return (
     <div>
       <ImageText text="AceUi"  backgroundImage="/image-text.jpg" />
    </div>
  );
}`}
</CopyCode>

## Variants

<CopyCode>
  {`// Different font sizes
<ImageText 
  text="Large Title" 
  backgroundImage="/hero-bg.jpg"
  fontSizeClasses="text-6xl md:text-8xl lg:text-9xl"
/>

// Custom alignment
<ImageText 
  text="Left Aligned" 
  backgroundImage="/texture.jpg"
  align="left"
/>

// Custom styling
<ImageText 
  text="Custom Style" 
  backgroundImage="/gradient.jpg"
  fontWeight="font-black"
  className="tracking-wider"
/>`}
</CopyCode>

## Features

-  Image background text masking effect
-  Responsive font sizing with Tailwind classes
-  Multiple text alignment options: `left`, `center`, `right`

## Notice

<InfoOut variant="warning">
Make sure your background images are optimized and have good contrast with surrounding elements for better readability and performance.
</InfoOut>
