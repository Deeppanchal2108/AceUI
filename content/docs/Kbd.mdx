## Usage

<CopyCode>
  {`import { Kbd } from "@/components/kbd";

  return (
    <div>

  );
}`}

## Variants

<CopyCode>
  {`// Default variant
<Kbd keys={["ctrl", "c"]} />

// Outline variant  
<Kbd keys={["shift", "enter"]} variant="outline" />

// Ghost variant
<Kbd keys={["escape"]} variant="ghost" />

// Different sizes
<Kbd keys={["tab"]} size="sm" />
<Kbd keys={["space"]} size="md" />
<Kbd keys={["command", "k"]} size="lg" />

// Single keys
<Kbd keys={["enter"]} />
<Kbd keys={["delete"]} />
<Kbd keys={["escape"]} />

// Multiple key combinations
<Kbd keys={["command", "shift", "p"]} />
<Kbd keys={["ctrl", "alt", "delete"]} />

// Arrow keys
<Kbd keys={["up"]} />
<Kbd keys={["down", "right"]} />

// Custom text keys
<Kbd keys={["F1"]} />
<Kbd keys={["a", "b", "c"]} />`}
</CopyCode>

## Features

- ⌨️ Realistic keyboard key styling with gradients and shadows
- 🔤 Support for special keys with proper Unicode symbols
- 📏 Multiple sizes: `sm`, `md`, `lg`
- 🎨 Three variants: `default`, `outline`, `ghost`
- 🌙 Dark mode support with proper contrast
- 🔗 Supports key combinations and sequences
- ♿ Accessible with proper semantic structure
- ⚙️ TypeScript support with predefined key types

## Notice

The component uses Unicode symbols for special keys which may not render consistently across all fonts and operating systems. Test thoroughly in your target environments.
