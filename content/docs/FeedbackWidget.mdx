## API Integration

Replace the mock API function with your actual endpoint:

<CopyCode>
  {`// Replace this mock function
const mockFeedbackAPI = () =>
  new Promise<void>((resolve, reject) => {
    setTimeout(() => {
      if (Math.random() > 0.9) {
        reject(new Error("Network error"));
      } else {
        resolve();
      }
    }, 1000 + Math.random() * 1000);
  });

// With your actual API call
const submitFeedbackAPI = async (satisfactionLevel: number, comment: string) => {
  const response = await fetch('/api/feedback', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({
      satisfactionLevel,
      comment,
      timestamp: new Date().toISOString(),
    }),
  });

  if (!response.ok) {
    throw new Error('Failed to submit feedback');
  }

  return response.json();
};`}
</CopyCode>

## Component Structure

The FeedbackWidget consists of several key parts:

1. **Header Section**: Contains the question text and emoji rating buttons
2. **Expandable Form**: Comment textarea and submit button (shows when rating selected)
3. **Success Message**: Animated confirmation with checkmark icon
4. **State Management**: Custom hook handling submission logic
