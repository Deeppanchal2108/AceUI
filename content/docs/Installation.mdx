<Steps>

### Create project

Run the `init` command to create a new Next.js project or to setup an existing one:

<CopyCode>
{`npx shadcn@latest init`}

</CopyCode>

### Add components

You can now start adding components to your project.

<CopyCode>
{`
npx shadcn@latest add "https://ace-ui-six.vercel.app/r/text-scramble.json"`}

</CopyCode>

### Import component

The command above will add the `TextScramble` component to your project. You can then import it like this:



<CopyCode>
{
  `
  import { TextScramble } from "@/components/ui/text-scramble"; 

export default function ScrambleExample() {
 return (
   <div className="flex justify-center items-center h-screen bg-gray-900 text-white">
     <TextScramble 
       className="text-blue-400"
       speed={30}
       duration={800}
     >
       Hover Over This Text
     </TextScramble>
   </div>
 );
}
  `
}


</CopyCode>

</Steps>
